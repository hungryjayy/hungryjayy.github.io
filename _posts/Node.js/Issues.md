## 200222

* 비동기 처리에 대한 고민

```javascript
func().then(async () => {
  if (foo()) {
   await promise() // first
  }
  await promise() // second
 })
```

​	이와 같은 코드에서 first → second 순서 보장. If() 조건절의 판별은 그 즉시 일어남
