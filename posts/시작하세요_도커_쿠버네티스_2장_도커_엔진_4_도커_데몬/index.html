мӢңмһ‘н•ҳм„ёмҡ”! лҸ„м»Ө / мҝ лІ„л„ӨнӢ°мҠӨ 2мһҘ лҸ„м»Ө м—”м§„ - 4. лҸ„м»Ө лҚ°лӘ¬

By hungryjayy
Posted Aug 4, 2022 4 min read


лҸ„м»Ө нҒҙлқјмқҙм–ёнҠёмҷҖ лҸ„м»Ө лҚ°лӘ¬

  • лҸ„м»Ө нҒҙлқјмқҙм–ёнҠё: Mac кё°мӨҖ /usr/local/bin/docker, лҰ¬лҲ…мҠӨ кё°мӨҖ /usr/bin/dockerм—җ мЎҙмһ¬. лҸ„м»ӨлҚ°лӘ¬мқҳ APIлҘј нҳём¶ңн•ЁмңјлЎңмҚЁ мң м Җк°Җ CLIлЎң лҸ„м»ӨлҘј мқҙмҡ©н•  мҲҳ мһҲлҸ„лЎқ н•ңлӢӨ. /var/run/docker.sock Unix мҶҢмј“мқ„ мқҙмҡ©н•ңлӢӨ.
  • лҸ„м»Ө м„ңлІ„(лҚ°лӘ¬ - dockerd): /usr/bin/dockerd (Macм—җм„ңлҠ” лҸ„м»Ө лҚ°мҠӨнҒ¬нғ‘м—җ мЎҙмһ¬). dockerd лӘ…л №м–ҙлҘј нҶөн•ҙ лҸ„м»Ө лҚ°лӘ¬мқ„ нҸ¬к·ёлқјмҡҙл“ң нҳ•нғңлЎң м§Ғм ‘ мӢӨн–үн•  мҲҳ мһҲлӢӨ.
    • --insecure-registry -> л Ҳм§ҖмҠӨнҠёлҰ¬ м»Ён…Ңмқҙл„Ҳ кө¬м¶• / --storage-opt -> мҠӨнҶ лҰ¬м§Җ л°ұм—”л“ң ліҖкІҪ л“ұ лӢӨм–‘н•ң лҸ„м»Ө лҚ°лӘ¬ м„Өм •мқҙ к°ҖлҠҘн•ҳкі , л§ӨлІҲ м§Ғм ‘ CLIлЎң м№  н•„мҡ” м—Ҷмқҙ м„Өм •нҢҢмқј DOCKER_OPTSм—җ мһ…л Ҙн•ҙлҶ“мқ„ мҲҳ мһҲлӢӨ.
    • k8sлЎң л”°м§Җл©ҙ Podmanм—җ лҢҖмқ‘лҗңлӢӨкі  ліҙл©ҙ лҗңлӢӨ.


лҸ„м»Ө лҚ°лӘ¬ м ңм–ҙ: -H

  • н•ҙлӢ№ мҳөм…ҳкіј н•Ёк»ҳ IP, portлҘј м Ғм–ҙл‘җл©ҙ мӣҗкІ© APIмқё Docker remote APIлЎң лҸ„м»ӨлҘј м ңм–ҙн•  мҲҳ мһҲлӢӨ. (HTTP мҡ”мІӯ)
    • л§Ңмқј remote docker daemon мЈјмҶҢл§Ң м Ғмңјл©ҙ н•ҙлӢ№ лҸ„м»Ө лҚ°лӘ¬м—җ лЎңм»¬м—җм„ң мӮ¬мҡ©н•ҳлҚҳ unix мҶҢмј“мқҙ л№„нҷңм„ұнҷ” лҗҳм–ҙ лЎңм»¬ CLIлҘј мӮ¬мҡ© н•  мҲҳ м—ҶкІҢ лҗҳлӢҲ, н•Ёк»ҳ мһ…л Ҙн•ҳл©ҙ л‘ҳлӢӨ мҡ”мІӯмқ„ л°ӣм•„ мІҳлҰ¬н•  мҲҳ мһҲлӢӨ.
    • к°Ғ м–ём–ҙлі„лЎң лқјмқҙлёҢлҹ¬лҰ¬л“Өмқҙ мҳӨн”ҲмҶҢмҠӨ нҳ•нғңлЎң мЎҙмһ¬н•ңлӢӨ. к°ңл°ңм—җ н•„мҡ”н•ң кІҪмҡ° мқҙмҡ©н•  мҲҳ мһҲлӢӨ.


лҸ„м»Ө лҚ°лӘ¬м—җ ліҙм•Ҳ : вҖ“tlsverify

: кё°ліём ҒмңјлЎң лҸ„м»Өм—җлҠ” ліҙм•Ҳ м—°кІ°мқҙ м„Өм •лҸјмһҲм§Җ м•ҠлҠ”лӢӨ. remote APIлҘј мӮ¬мҡ©н•  л•ҢлҸ„ л§Ҳм°¬к°Җм§Җмқҙкё° л•Ңл¬ём—җ TLSлҘј м Ғмҡ©н•ҙм•јн•ңлӢӨ.


мҠӨнҶ лҰ¬м§Җ л“ңлқјмқҙлІ„ ліҖкІҪ: вҖ“storage-driver

: лҸ„м»ӨлҠ” мҠӨнҶ лҰ¬м§Җ л°ұм—”л“ң кё°мҲ мқ„ мқҙмҡ©н•ҙ м»Ён…Ңмқҙл„ҲмҷҖ мқҙлҜём§ҖлҘј м ҖмһҘ л°Ҹ кҙҖлҰ¬н•ңлӢӨ. Macм—җм„ңлҠ” overlay2.

  • мҠӨлғ…мғ·мқҳ к°ңл…җмқҙ мқҙмҡ©лҗңлӢӨ: мӣҗліёмқҖ мқҪкё°м „мҡ©мңјлЎң мӮ¬мҡ©н•ҳкі  ліҖкІҪмқҖ мғҲлЎңмҡҙ кіөк°„м—җ н• лӢ№н•ҳкё°
    • CoW: мҠӨлғ…мғ· кіөк°„м—җ мӣҗліёнҢҢмқј ліөмӮ¬ нӣ„ ліҖкІҪмӮ¬н•ӯ м“°кё°. (write мҳӨлІ„н—Өл“ң 2нҡҢ)
    • RoW: мӣҗліё мҠӨлғ…мғ·мқ„ freeze мӢңнӮЁ л’Ө мҠӨлғ…мғ· кіөк°„м—җ ліҖкІҪлҗң мӮ¬н•ӯ мғҲлЎӯкІҢ н• лӢ№л°ӣм•„ м“°кё°(write 1нҡҢ)
  • мқҙлҜём§Җ л Ҳмқҙм–ҙлҠ” к°Ғ мҠӨлғ…мғ·м—җ н•ҙлӢ№н•ңлӢӨ. м»Ён…Ңмқҙл„Ҳ л Ҳмқҙм–ҙмқҳ ліҖкІҪм җмқ„ м»Өл°Ӣн•ҳл©ҙ ліҖкІҪмӮ¬н•ӯ(diff)к°Җ мҠӨлғ…мғ·(commit)мңјлЎң мғқм„ұлҗҳкі  н•ҙлӢ№ diffлҠ” н•ҳлӮҳмқҳ мқҙлҜём§Җ л Ҳмқҙм–ҙлЎңм„ң мЎҙмһ¬н•ңлӢӨ.


лҚ°лӘ¬ лӘЁлӢҲн„°л§Ғ

: м»Ён…Ңмқҙл„Ҳ мғҒнғң лҝҗ м•„лӢҲлқј лҸ„м»ӨлҘј PaaSлЎң м ңкіөн•ҳлҠ” кІҪмҡ°лҘј мң„н•ҙ лҸ„м»Ө лҚ°лӘ¬ мһҗмІҙлҘј лӘЁлӢҲн„°л§Ғ н•  мҲҳ мһҲлӢӨ.


  • events: м»Ён…Ңмқҙл„Ҳ, мқҙлҜём§Җ, ліјлҘЁ л„ӨнҠёмӣҢнҒ¬ кҙҖл Ё лӘ…л №м–ҙ мӢӨн–ү кІ°кіјлҘј лӘЁлӢҲн„°л§Ғ
  • stats: м»Ён…Ңмқҙл„Ҳ мһҗмӣҗ мӮ¬мҡ©лҹү лӘЁлӢҲн„°л§Ғ
  • system df: м»Ён…Ңмқҙл„Ҳ, мқҙлҜём§Җ, ліјлҘЁ мӮ¬мҡ©лҹүкіј мӮӯм ңн•ЁмңјлЎңмҚЁ нҷ•ліҙ к°ҖлҠҘн•ң кіөк°„ л“ұ лӘЁлӢҲн„°л§Ғ -> prune кҙҖл Ё
  • CAdvisor: нҳёмҠӨнҠёмқҳ лҸ„м»Ө кҙҖл Ё л””л үн„°лҰ¬лҘј лӘЁл‘җ н•ҙлӢ№ м»Ён…Ңмқҙл„ҲмҷҖ ліјлҘЁ кіөмң н•ңлӢӨ. лҰ¬мҶҢмҠӨ, м»Ён…Ңмқҙл„Ҳ мғҒнғң л“ұмқ„ нҷ•мқё н•  мҲҳ мһҲлӢӨ. нҳёмҠӨнҠёмқҳ лҸ„м»Ө лҰ¬мҶҢмҠӨлҘј кіөмң н•ҳлҠ” кІғмқҙкё° л•Ңл¬ём—җ лӢЁмқј нҳёмҠӨнҠё н•ңм •мқҙлӢӨ.
